The Grid Skyscrapers: Puzzle

A logic puzzle fusing Sudoku with urban architecture. Fill a 9×9 grid where each number is a building height — but edge clues reveal only how many skyscrapers are visible from each vantage point, since taller structures block shorter ones behind them. Four difficulty tiers from relaxed to expert, with a 3-lives system that makes every placement count.
